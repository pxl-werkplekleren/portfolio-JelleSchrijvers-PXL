Reflectie PE: C# Console

Programmeren - 19/11/2025

Beschrijving

In deze PE: C# Console werkte ik aan verschillende opdrachten, waaronder Trader-02 en Trader-03, waarbij ik mijn C# kennis moest toepassen in praktische scenario's. Het doel was om mijn programmeervaardigheden te testen, logica correct toe te passen en inzicht te krijgen in best practices door mijn oplossing te vergelijken met die van de lectoren.

Persoonlijke uitdagingen

Een belangrijke uitdaging was dat ik sommige logische structuren te ingewikkeld schreef. Bij Trader-02 bijvoorbeeld, gebruikte ik een if-structuur die veel langer en complexer was dan nodig. Hoewel mijn code functioneel correct was, was deze minder efficiënt dan de modeloplossing van de lectoren. Bij Trader-03 bleek dat ik de random waarden voor FuelForMetalRate verkeerd had ingesteld en de 50/50-kansstructuur niet logisch klopte. Dit liet zien dat ik sommige basisconcepten nog beter moet beheersen en bewust moet nadenken over de logica van mijn code. Ook merkte ik dat ik soms output vergat of geen herhaling implementeerde bij foutieve input, waardoor mijn programma niet altijd robuust was. Het vergelijken met de modeloplossing hielp om deze fouten te herkennen.

Bijgeleerd

Door deze PE heb ik geleerd dat eenvoud en overzicht in code vaak beter zijn dan te complexe structuren. Ik begrijp nu beter hoe ik if-structuren kort en logisch kan opbouwen. Ook heb ik geoefend met het genereren van random waarden en het correct toepassen van voorwaarden in C#. Verder realiseerde ik mij het belang van systematisch controleren van alle invoer en het nakijken van de output om logische fouten te voorkomen. Het bekijken van de modeloplossing gaf inzicht in best practices en hoe je efficiënt en overzichtelijk code schrijft. Tot slot heb ik mijn kennis van klasse-oproepen uitgebreid, bijvoorbeeld bij het koppelen van invoer aan nieuwe TraderStations, wat mijn begrip van objectgeoriënteerd programmeren in C# versterkt. Mijn plan is om deze inzichten in toekomstige opdrachten actief toe te passen en basisconcepten verder te oefenen.
