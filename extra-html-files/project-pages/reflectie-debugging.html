Reflectie PE: Debugging (C#)

Programmeren - 05/12/2025

Beschrijving

In deze PE: Debugging (C#) moest ik een bestaande applicatie analyseren en de aanwezige bugs opsporen en oplossen met behulp van Visual Studio. Het doel was om inzicht te krijgen in foutopsporing, logisch denken en het correct toepassen van C#-concepten in een echte codebase. Een deel van de applicatie werkte correct, terwijl het andere gedeelte voortdurend vastliep, waardoor het lastig was om het volledige programma te testen. Buiten de lessen oefende ik met het maken van eigen projecten en het systematisch opsporen van fouten om ervaring op te doen met debugging.

Persoonlijke uitdagingen

Tijdens het debuggen merkte ik dat ik nog niet volledig zelfstandig bugs kan analyseren en oplossen. Sommige problemen waren moeilijk te vinden en te begrijpen, vooral wanneer meerdere fouten tegelijk aanwezig waren. Daarnaast moest ik leren omgaan met frustratie wanneer het programma niet werkte zoals verwacht, en geduld oefenen bij het doorlopen van de foutopsporingsstappen.

Bijgeleerd

Door deze PE heb ik mijn vaardigheden in foutopsporing in Visual Studio versterkt. Ik leerde systematisch mijn eigen code controleren, logische stappen volgen en kleinere testcases maken om fouten te isoleren. Ik maakte gebruik van hulpmiddelen zoals YouTube-tutorials en online codingwebsites om mijn C# kennis bij te schaven. Hoewel ik nog niet volledig zelfstandig bugs kan oplossen, heb ik inzicht gekregen in hoe ik stap voor stap problemen kan analyseren en aanpakken. Deze ervaring zal mij helpen bij toekomstige opdrachten en projecten, omdat ik nu beter begrijp hoe ik mijn code kan structureren en controleren om fouten sneller te identificeren.
