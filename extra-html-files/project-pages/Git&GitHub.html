Werkplekleren 1 - 16/10/2025

Beschrijving

In deze opdracht heb ik een eenvoudige website gemaakt met HTML en CSS, verdeeld over twee pagina’s: index.html en about.html. De startpagina bevatte een overzicht van veel gemaakte fouten bij Git & GitHub en een link naar de "about"-pagina met mijn naam. De focus lag op het correct toepassen van Git-commando’s, commits en het pushen van het resultaat naar een GitHub repository.

Persoonlijke uitdagingen

Technisch gezien had ik de code correct geschreven, maar mijn fout was dat ik alles in één keer committeerde in plaats van per vereiste stap. Dit maakte het moeilijker om de voortgang overzichtelijk te houden en te controleren welke wijzigingen bij welke stap hoorden. Voor volgende projecten wil ik leren om **per kleine stap een commit te maken met een duidelijke message**, zodat het overzichtelijker wordt en fouten sneller te traceren zijn.

Bijgeleerd

Door deze opdracht heb ik geleerd hoe belangrijk het is om Git gestructureerd te gebruiken: commits per stap maken, duidelijke messages toevoegen en regelmatig pushen. Ik heb ook praktische ervaring opgedaan met het aanmaken van meerdere HTML-pagina’s, het linken van pagina’s via hyperlinks en het correct opzetten van een repository. Dit zal me helpen bij toekomstige projecten om overzicht te behouden en fouten in versiebeheer te vermijden.
